# MALLOC

The goal of this project is to recode [Malloc's function], [Free's function] and [Realloc's function]

# Language
  - C Programming

### Run

```sh
$ make
$ LD_PRELOAD=./libmy_malloc.so "cmd" ("cmd" is the command, try "ls")
```
